HTML Reference

<p>

<p> is used to define paragraphs
eg: <p>This is a paragraph</p>

<span>

<span> is a inline container used to mark up a part of a text or a part of a document.
eg: <p>mark up a <span>part</span>of a text</p>

<strong>

The <strong> tag is used to define text with strong importance.
eg: <p><strong>Important:</strong>Before proceeding ,make sure you add plenty of butter</p>

<em>

The <em> tag is used to define emphasized text. The content inside is typically displayed in italic.
eg: <p>We <em>cannot</em> live like this.</p>

<small>

The <small> element represents side-comments and small print,like copyright and legal text,independent of its styled presentation.
eg: <p>This is a content.<small>The content is licensed under a Generic license.</small></p>

<b>

The <b> tag specifies bold text without any extra importance.
eg: <p>This is <b>bold</b>text</p>

<i>

The <i> tag defines a part of text in an alternative voice or mood. The content is typically is displayed in italic. The tag is often used to indicate a technical term, a phrase from another language , a thought , a ship nane ,etc.
eg: <p><i>Lorem ipsum</i> is the most popular filler text in history</p>

<u>

The <u> element represents a span of inline text which should be rendered in a way that it indicates that it has a non-textual annotation.
eg: <p>You could use this element to highlight <u>speling</u> mistakes.</p>

<s>

The <s> element represents things that are no longer relevant or no longer accurate.
eg: <p><s>Only 50 tickets left!</s></p>
<p>SOLD OUT</p>

<sup>

The <sup> element specifies inline text which is to be displayed as superscript for solely typographical reasons.
eg: a<sup>2</sup>

<sub>

The <sub> tag defines subscript text. Subscript text can be used for chemical formulas.
eg: <p>H<sub>2</sub>O</p>

<abbr>

The <abbr> tag defines an abbreviation or an acronym.
eg: <p>The <abbr title="World Health Organization">WHO</abbr> was founded in 1948</p>

<cite>

The <cite> tag defines the title of a creative work.
eg: <p><cite>The Scream</cite> by Edward Munch. Painted in 1893.</p>

<q>

The <q> tag indicates that the enclosed text is a short inline quotation.
eg: <p>He said <q>It is a good day</q></p>

<code>

The <code> tag is used to define a piece of computer code.
eg: <p><code>push()</code></p>

<kbd>

The <kbd> tag represents a span of inline text denoting textual user input from a keyboard, voice input,or any other text entry device.
eg: <p>Please press<kbd>ctrl</kbd></p>

<samp>

The <samp> tag is used to define sample output from a computer program.
eg: <p><samp>File not found</samp></p>

<var>

The <var> tag is used to define a variable in programmin or in a mathematical expression.
eg: <p>The volume of a box is <var>x</var></p>

<mark>

The <mark> tag defines text that should be marked or highlighted.
eg:<p> Do not forgot to buy <mark>milk</mark> today</p>

<del>

The <del> tag defines text that has been deleted from a document.
eg: <p>My favourite colour is <del>blue</del> <ins>red</ins>!</p>

<ins>

The <ins> tag defines a text that has been inserted into a document.
eg: <p>My favourite colour is <del>blue</del> <ins>red</ins>!</p>

<time>

The <time> tag defines a specific time (or date time). The datetime attribute of this element is used to translate the time into a machine-readable format.
eg: <p>Open from <time>10:00</time> every weekday</p>

<dfn>

The <dfn> tag stands for the definition element, and it specifies a term that is going to be defined within the content.
eg: <p><dfn>HTML</dfn> is the standard markup language for creating web pages</p>
